两栏布局实现

Apr 3, 2020 | 技术 | 阅读

整理几种实现两栏布局(一栏固定宽度，一栏自适应)的方法，方便以后查看
flex 布局、position:absolute、float、inline-block、grid、table

flex 布局

表现

  • 在不设置高度的情况下，两栏的实际高度 以两者的最大高度 决定（两栏的底部是对齐的）
  • [main的高度] 由两者的最大高度决定
  • [屏幕宽度少于200px] son2表现为可展示的最小宽度（一个字体的宽度）
  • 如果不对son1 设置flex-shrink:0的话，son1会缩放（son1的实际宽度少于200px）

position:absolute

表现

  • 在不设置高度的情况下，两栏的实际高度 以自身的内容 决定（两栏的底部是不对齐的）
  • [main的高度] 由son2决定
  • [屏幕宽度少于200px] son2表现为宽度为0

float

表现

  • 在不设置高度的情况下，两栏的实际高度 以自身的内容 决定（两栏的底部是不对齐的）
  • [main的高度] 由son2决定 （所以这里要注意清除浮动）
  • [屏幕宽度少于200px] son2表现为宽度为0

inline-block

原理： 将两栏变成inline-block，并列排放

表现

  • 在不设置高度的情况下，两栏的实际高度 以自身的内容 决定（两栏的底部是不对齐的）
  • [main的高度] 由两者的最大高度决定
  • [屏幕宽度少于200px] son2被挤到下一行，但其宽度是0(因为100%-200px是个负数？)

grid

表现

  • 在不设置高度的情况下，两栏的实际高度 以两者的最大高度 决定（两栏的底部是对齐的）（高度差值用padding补充）
  • [main的高度] 由两者的最大高度决定
  • [屏幕宽度少于200px] son2表现为可展示的最小宽度（一个字体的宽度）不会被缩放

    table

表现

  • 在不设置高度的情况下，两栏的实际高度 以两者的最大高度 决定（两栏的底部是对齐的）
  • [main的高度] 由两者的最大高度决定
  • [屏幕宽度少于200px] son2表现为可展示的最小宽度（一个字体的宽度）不会被缩放
