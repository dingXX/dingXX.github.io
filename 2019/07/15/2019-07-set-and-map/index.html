set

不重复的值的集合，成员可以是任何类型的数据

  • 值不会发生类型转换，判断两个值是否不同用Same-value-zero equality（类似于===，但NaN的判断为相同）
  • 两个NaN值认为相同
  • 两个物理地址不一致对象，认为是不一样
  • 没有键名，只有键值(或者说键名和键值是同一个值)

方法

  • 构造函数Set()：接受一个数组（或者具有 iterable 接口的其他数据结构）作为参数，用来初始化。返回Set实例
  • size: Set实例的成员个数
  • add(value): 添加值，返回Set实例
  • delete(value): 删除值，返回布尔值，表示是否删除成功（存在并删除为true，不存在返回fasle）
  • has(value): 是否为Set实例的成员
  • clear(): 清除所有成员，无返回值undefined

遍历相关的方法（Set的遍历顺序就是插入顺序）：

  • keys(): 返回键名的遍历器
  • values(): 返回键值的遍历器（表现和keys()一致）
  • entries(): 返回键值对的遍历器
  • forEach(): 使用回调函数遍历每个成员 (value,key)都是同一个值

WeakSet

与Set 类似，不重复的值的集合，但成员只能是对象。

  • WeakSet 中的对象都是弱引用，即垃圾回收机制不考虑 WeakSet 对该对象的引用（对象在外部消失，它在 WeakSet 里面的引用就会自动消失）
  • WeakSet 不可遍历（WeakSet 内部有多少个成员，取决于垃圾回收机制有没有运行，运行前后很可能成员个数是不一样的，而垃圾回收机制何时运行是不可预测的）

方法

  • 构造函数WeakSet: 接受一个数组（或者具有 iterable 接口的其他数据结构）作为参数，用来初始化。返回WeakSet实例
  • add(value): 新增成员，返回WeakSet实例
  • delete(value): 删除成员，返回布尔值
  • has(value): 判断成员是否存在，返回布尔值

Map

类似于Object，键值对集合。但是“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键。

  • Object的“键”只能是字符串
  • 对象类的键名以物理地址作为是否相同的判断
  • NaN作为键名，认为是相同键名

方法

  • 构造函数：可以接受一个数组作为参数。该数组的成员是一个个表示键值对的数组。
  • size: 键值对个数
  • set(key,value): 设置键值对，返回Map实例
  • get(key): 根据键名获取键值
  • has(key): 某个键是否在当前 Map 实例之中
  • delete(key): 返回布尔值，删除某个键值对
  • clear(): 清空Map实例中所有成员，没有返回值。

可被遍历，对应的遍历方法

  • keys(): 返回键名的遍历器
  • values(): 返回键值的遍历器
  • entries(): 返回所有成员的遍历器。
  • forEach(): 遍历 Map 的所有成员

WeakMap

与Map类似，键值对集合。但键名只能是对象（null除外）

  • 键名只能是对象，不能用null作为键名
  • 键名所指向的对象是弱引用，不计入垃圾回收机制（同理WeakSet）
  • 键值所指向的对象是正常引用。（外部消除了对象的引用，但WeakMap 内部的对象引用依然存在）
  • 不可遍历

方法

无size属性，不可遍历

  • 构造函数：可以接受一个数组作为参数。该数组的成员是一个个表示键值对的数组。
  • set(key,value): 设置键值对，返回Map实例
  • get(key): 根据键名获取键值
  • has(key): 某个键是否在当前 Map 实例之中
  • delete(key): 返回布尔值，删除某个键值对
